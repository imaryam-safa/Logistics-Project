# Logistics-Project
A publically available logistics report is used to create an end-to-end project using MS SQL Server and then visualized in Power BI. Additionally, a regression model is used to identify driver churn. 

SQL SERVER: 
1. The database files are loaded into the MS SQL Server and the database diagram created. 
2. Various SQL queries are performed to identify driver performance (On-time rates, MPG, revenue per mile) in comparison to their tenure as well.
3. A new table was also created to consolidate driver performance.

POWER BI
1. The database was connected to Power BI and the data was visualized.

PYTHON/JUPYTER NOTEBOOK:
1. The data was loaded into the pandas dataframe, it was cleaned and manipulated.
2. A logistic regression model was created to identify driver churn.
